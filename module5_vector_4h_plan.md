**17:45-18:10 | 词汇语义学基本概念 (25分钟)**

- [ ] **对比原则**: 不同形式 → 总有语义差异

**18:10-18:30 | 相似性 vs 相关性 (20分钟)**

  - cup/mug: 高相似
- [ ] **相关性（Relatedness）**: 共同出现/某种关系
  - coffee & cup: 相关但不相似

- [ ] **语义场（Semantic Field）**: 同领域词集合
- [ ] **内涵（Connotation）**: 情感色彩

### 📖 内容与时间分配

**22:00-22:25 | 词-语境矩阵 (25分钟) ⚠️核心**
- [ ] **矩阵结构**:
  - 行 = 目标词（target words）
  - 列 = 语境词（context words）
  - 单元格 = 共现次数
- [ ] **语境窗口**: 通常 ±4 词
- [ ] **构建示例**:
  ```
  "cherry pie" → cherry向量 = [1, 0, 1]
  对应 [a, computer, pie] 的共现次数
  ```
- [ ] **这就是词嵌入的一种实现方式！**
- **练习**: 手工构建一个3x3的共现矩阵

**22:25-22:50 | 余弦相似度 (25分钟) ⚠️核心**
- [ ] **点积 vs 余弦**:
  - 点积: $\vec{v} \cdot \vec{w} = \sum v_i w_i$
  - 余弦: $\frac{\vec{v} \cdot \vec{w}}{|\vec{v}||\vec{w}|}$
- [ ] **为什么用余弦**: 消除向量长度影响
- [ ] **几何直觉**:
  - θ = 0° → cos = 1 → 最相似
  - θ = 90° → cos = 0 → 无相似性
  - θ > 90° → cos < 0 → 负相关
- [ ] **核心规律**: 夹角越小 → 相似度越高
- **练习**: 手算两个简单向量的余弦相似度

**22:50-23:00 | 稀疏向量问题 (10分钟)**
- [ ] **Sparse Vectors问题**:
  - 维度 = 词汇表大小（非常大）
  - 大量0（大多数词对不共现）
  - 计算和存储成本高
- [ ] **引出Word2Vec**: 下一段重点
- **思考**: 如何解决稀疏性问题？

## 📝 重点公式

### 余弦相似度
$$\text{cosine\_sim}(\vec{v}, \vec{w}) = \frac{\vec{v} \cdot \vec{w}}{|\vec{v}||\vec{w}|}$$

### 向量长度（范数）
$$|\vec{v}| = \sqrt{\sum_i v_i^2}$$
