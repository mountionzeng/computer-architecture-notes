## 📅 第1小时: 神经网络基础 (00:00-60:00)

### 🎯 学习目标
- 理解神经网络的本质（非线性函数组合）
- 掌握神经单元的组成和计算
- 理解激活函数的作用
- 解决XOR问题的思路

### 📖 内容 (第1-24页)

**00-20分钟: 基础概念**
- [ ] 什么是神经网络？(第3-5页)
  - 非线性函数的组合
  - Universal Approximation Theory
- [ ] 神经网络组件 (第6-7页)
  - Nodes (节点)
  - Weights (权重)
  - Biases (偏置)
  - Activation Function (激活函数)
  - Layers (层)

**20-40分钟: 神经单元计算**
- [ ] 线性单元：加权和 (第8页)
  - z = w·x + b
- [ ] 激活函数详解 (第9-11页)
  - **Sigmoid**: σ(z) = 1/(1+e^(-z))
  - **Tanh**: 双曲正切
  - **ReLU**: max(0, z)
- [ ] **实战演练** (第12-14页)
  - 手算神经单元输出
  - 给定: w=[0.2,0.3,0.9], b=0.5, x=[0.5,0.6,0.1]
  - 计算 z 和 y

**40-60分钟: XOR问题**
- [ ] 感知机的局限 (第15-18页)
  - AND/OR可计算
  - XOR不可线性分割
- [ ] 多层网络解决方案 (第19-24页)
  - 隐藏层创建新表示
  - 手工验证XOR网络计算
  - **练习**: 验证输入[1,0]和[1,1]

### ✅ 小结检查点
- [ ] 能手算神经单元输出
- [ ] 能解释为什么XOR需要多层
- [ ] 理解激活函数的非线性作用

## 📅 第2小时: 前馈神经网络架构 (60:00-120:00)

### 🎯 学习目标
- 理解前馈网络的层次结构
- 掌握矩阵表示法
- 了解实际应用案例

### 📖 内容 (第25-38页)

**60-80分钟: 网络架构**
- [ ] Feedforward架构 (第25页)
  - 信息单向流动
- [ ] 层类型 (第26页)
  - **Dense** (全连接): 矩阵乘法
  - **Convolutional** (卷积): 卷积核
  - **Recurrent** (循环): 状态传递
  - Pooling / Dropout
- [ ] 全连接网络 (第27页)
  - 每层的每个节点都连接到下一层的所有节点

**80-100分钟: 数学表示**
- [ ] 矩阵表示法 (第28-29页)
  - h = σ(Wx + b)
  - y = softmax(Uh + c)
- [ ] **Softmax函数** (第30页)
  - 多分类输出
  - 概率归一化
- [ ] 深度网络通用记号 (第31页)
- [ ] **为什么需要非线性?** (第32页)
  - 多层线性 = 单层线性

**100-120分钟: 实际应用**
- [ ] 情感分类架构 (第33-34页)
  - 文本 → 特征 → 分类
- [ ] 从特征到嵌入 (第36页)
- [ ] 神经语言模型 (第37-38页)
  - 预测下一个词
  - 架构示例

### ✅ 小结检查点

- [ ] 理解矩阵表示的维度
- [ ] 知道Softmax用于什么场景

## 📅 第3小时: 训练与反向传播 (120:00-180:00)

### 🎯 学习目标
- 理解Loss函数和梯度下降
- 掌握反向传播算法
- 了解优化技巧

### 📖 内容 (第39-62页)

**120-145分钟: 训练基础**
- [ ] 训练三要素 (第39-41页)
  - Loss (损失函数)
  - Gradient Descent (梯度下降)
  - Backpropagation (反向传播)
- [ ] **Cross-Entropy Loss** (第42-43页)
  - L = -Σ y_true · log(y_pred)
  - 为什么用交叉熵?
- [ ] Gradient Descent (第44页)
  - θ = θ - α·∇L

**145-165分钟: 反向传播详解**
- [ ] **Chain Rule** (第46页)
  - 复合函数求导
- [ ] 反向传播示例 (第47-49页)
  - Forward pass计算
  - Backward pass计算梯度
- [ ] 激活函数的导数 (第48页)
  - σ'(z) = σ(z)·(1-σ(z))
  - ReLU'(z) = 1 if z>0 else 0
- [ ] **实战演练** (第50-52页)
  - 手算简单网络的梯度
  - Loss对参数的导数
  - 隐藏层梯度

**165-180分钟: 优化技巧与总结**
- [ ] 权重初始化 (第54页)
  - 避免梯度消失/爆炸
- [ ] Dropout正则化 (第55页)
  - 随机丢弃节点防止过拟合
- [ ] 超参数 (第56页)
  - Learning rate, batch size等
- [ ] 优化算法 (第57页)
  - **Mini-batch**: 小批量更新
  - **Momentum**: 动量加速
  - **Adam**: 自适应学习率(最常用)
- [ ] 现代框架 (第58页)
  - PyTorch / TensorFlow
  - 自动求导 + GPU加速
- [ ] 课程总结 (第59-61页)
  - 关键概念回顾
  - NLP应用
  - 训练要点

### ✅ 小结检查点
- [ ] 能解释反向传播的原理
- [ ] 知道为什么用Adam而不是普通SGD
- [ ] 理解Dropout如何防止过拟合
