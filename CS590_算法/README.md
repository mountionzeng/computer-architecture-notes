# CS590 算法设计与应用

本目录整理了 CS590 课程的算法笔记、可视化演示与补充材料，目标是用于复习、查阅和快速回顾核心算法思想。

## 快速入口

- 课程总索引：`CS590_00_Course_Overview_Index.md`
- 复习速通：`Module 8-13 速通复习卡.md`
- 前置理解：`Module 8-13 前置理解版.md`

## 内容结构

- `Module 1` - `Module 13`：按课程模块整理的主笔记
- `*.html`：算法过程可视化（可在浏览器打开）
- `*.png` / `*.svg`：对应静态图示
- `补充材料/`：拓展知识与额外示例

## 推荐阅读顺序

1. `Module 1: Algorithm Analysis.md`
2. `Module 2: Basic Data Structures.md`
3. `Module 3: Binary Search Trees.md`
4. `Module 4: Priority Queues and Heaps.md`
5. `Module 5: Hash Tables.md`
6. `Module 6: Union-Find Structures.md`
7. `Module 7: Sorting and Selection.md`
8. `Module 8` - `Module 13`

## 使用说明

- 纯笔记阅读：直接打开 `.md` 文件即可。
- 演示交互：打开对应 `.html` 文件查看算法动态过程。
- 图示引用：`.png` / `.svg` 可直接用于报告或复习卡片。
